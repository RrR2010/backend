# TASK: Implement Global Session Revocation UseCase

---

## DESCRIPTION

Create use case to revoke all sessions for the current user (logout everywhere).

- Summary (1 sentence):
  Implement RevokeAllSessionsUseCase to invalidate all sessions for a user (logout from all devices).
- Steps (max 5):
  1. Create RevokeAllSessionsUseCase class
  2. Query all sessions for user
  3. Mark all as revoked in database
  4. Add DELETE /auth/sessions endpoint (no ID)
  5. Return count of revoked sessions
