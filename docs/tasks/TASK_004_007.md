# TASK: Implement Platform Admin Session Management

---

## DESCRIPTION

Add admin endpoints to allow Platform Admin to manage any user sessions.

- Summary (1 sentence):
  Implement Platform Admin capabilities to view and revoke any user sessions system-wide.
- Steps (max 5):
  1. Create AdminListSessionsUseCase for any user
  2. Create AdminRevokeSessionUseCase for any user
  3. Add GET /admin/users/:userId/sessions endpoint
  4. Add DELETE /admin/users/:userId/sessions/:sessionId endpoint
  5. Add authorization check for Platform Admin role
