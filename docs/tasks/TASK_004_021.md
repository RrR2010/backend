# TASK: Simplify Authentication Guards and Context Resolution

---

## DESCRIPTION

Reduce conditional guard branching by moving context resolution into a single predictable auth layer.

- Summary (1 sentence):
  Refactor the current guard setup so the backend resolves auth scope and tenant context in one place instead of spreading conditional logic across multiple guards.
- Steps (max 5):
  1. Review the current JWT and tenant context guards
  2. Consolidate context resolution into a single auth context pathway
  3. Keep guard behavior explicit and fail closed
  4. Remove legacy branching that exists only to support the temporary login workaround
  5. Ensure platform and tenant requests still pass through the same request pipeline shape
