## 2. DESCRIPTION

<!-- What needs to be done -->

- Summary (1 sentence): Convert normalized permission specs into CASL abilities for platform and tenant scopes.
- ## Steps (max 5):

- Attach tenantId conditions when the active scope is tenant.
- Attach ownership conditions when the permission requires self access.
- Return a buildable ability object that the guard can evaluate.
