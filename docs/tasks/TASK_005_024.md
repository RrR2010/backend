## 2. DESCRIPTION

<!-- What needs to be done -->

- Summary (1 sentence): Build the guard that reads route metadata, resolves the active scope, and blocks unauthorized requests by default.
- ## Steps (max 5):
- Read required permissions from handler metadata.
- Resolve whether the request is platform-scoped or tenant-scoped.
- Call the resolver and ability factory in the correct order.
- Reject the request when any required permission fails.
