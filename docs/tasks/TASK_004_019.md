# TASK: Refactor Token Issuance and Session Setup by Auth Scope

---

## DESCRIPTION

Consolidate access/refresh token issuance so platform and tenant sessions follow the same session setup rules.

- Summary (1 sentence):
  Extract token issuance and cookie setup into a single flow that handles platform and tenant sessions without duplicated controller branches.
- Steps (max 5):
  1. Extract token issuance into a dedicated service or helper
  2. Support platform-scoped and tenant-scoped session setup
  3. Persist refresh token/session data consistently for both scopes
  4. Set cookies with a single contract and consistent expiration values
  5. Remove duplicated cookie and token logic from the controller
