# TASK: Update Login Flow to Issue Refresh Token

---

## DESCRIPTION

Modify the login flow to issue refresh tokens alongside the pre-auth token.

- Summary (1 sentence):
  Update LoginUseCase and AuthController to issue refresh tokens during the login flow.
- Steps (max 5):
  1. Modify LoginUseCase to call RefreshTokenService
  2. Update login response to include refresh token (or set HttpOnly cookie)
