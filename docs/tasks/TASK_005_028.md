## 2. DESCRIPTION

<!-- What needs to be done -->

- Summary (1 sentence): Ensure tenant-scoped policies are backed by tenant-aware data access so cross-tenant leakage cannot occur through repositories.
- ## Steps (max 5):
- Review the repository methods used by tenant-scoped features.
- Require tenantId filtering on reads and writes where the current scope is tenant.
- Preserve the tenant check even when the guard already allowed the request.
- Document the invariant that authorization and data filtering are both required.
