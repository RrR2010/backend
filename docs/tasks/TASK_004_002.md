## DESCRIPTION

Create service to manage refresh token issuance, rotation, and validation.

- Summary (1 sentence):
  Implement the RefreshTokenService to handle refresh token creation, storage, rotation, and validation against revoked tokens.
- Steps (max 5):
  1. Create RefreshTokenService class in domain layer
  2. Implement generateRefreshToken() method with secure random generation
  3. Implement saveRefreshToken() to store hashed token in database
  4. Implement validateRefreshToken() to verify token validity
  5. Implement rotateRefreshToken() to invalidate old and create new token
